  • TorchCoin (TCH) is a cryptocurrency backed by the training of deep learning models written in PyTorch on the blockchain.

    Training PyTorch models is the mechanism behind mining TorchCoin.
    Verified deep learning researchers may submit their models to be trained on the blockchain.
    Miners use GPUs to train submitted PyTorch models for a TorchCoin reward.

  • Getting started


    For deep learning researchers

    First, install the TorchCoin model submission tools...
    Submit your models using:

    {% highlight bash %} $ tcsubmit [git_clone_url] [dataset_url] {% endhighlight %}

    For example:

    {% highlight bash %} $ tcsubmit https://github.com/nick11roberts/PGCnet.git \ https://drive.google.com/open?id=0BzHWFkitR5d_VjJfS1ZmMDV5RmM {% endhighlight %}

    For miners

    First, install the TorchCoin mining software...
    Start the mining process by:

    {% highlight bash %} $ tcsubmit [git_clone_url] [dataset_url] {% endhighlight %}

    For traders

    First, install the TorchCoin wallet software...
    Transfer TorchCoin using:

    {% highlight bash %} $ tcsubmit [git_clone_url] [dataset_url] {% endhighlight %}
