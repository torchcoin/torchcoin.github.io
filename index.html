  • TorchCoin (TCH) is a cryptocurrency backed by the training of deep learning models written in PyTorch on the blockchain.

    Training PyTorch models is the mechanism behind mining TorchCoin.
    Verified deep learning researchers may submit their models to be trained on the blockchain.
    Miners use GPUs to train submitted PyTorch models for a TorchCoin reward.


    Why?

    GPU resources are scarce due to their growing popularity in the cryptocurrency mining community, and for this reason among others, deep learning researchers often have difficulty obtaining GPU resources for their research. Training deep learning models as the mechanism for mining cryptocurrency is beneficial to everyone involved.


    Why PyTorch?

    Yo Tensorflow, I'm really happy for you, I'll let you finish, but PyTorch has one of the best dynamic computation graph engines of all time.

  • Getting started


    For deep learning researchers

    First, install the TorchCoin model submission tools...
    Submit your models using:

    {% highlight bash %} $ tcsubmit --git_clone_url [git_clone_url] \ --dataset_url [dataset_url] {% endhighlight %}

    For example:

    {% highlight bash %} $ tcsubmit https://github.com/nick11roberts/PGCnet.git https://drive.google.com/open?id=0BzHWFkitR5d_VjJfS1ZmMDV5RmM {% endhighlight %}

    For miners

    First, install the TorchCoin mining software...
    Start the mining process by:

    {% highlight bash %} $ tcminer --wallet_address [wallet_address] \ --pool_address [pool_address] {% endhighlight %}

    For example:

    {% highlight bash %} $ tcminer 6xkabhmf28hr7o2v1pgo8r7nq5 w1bbdurg7u2clgl5mujg1pp95t {% endhighlight %}

    For traders

    First, install the TorchCoin wallet software...
    Transfer TorchCoin using:

    {% highlight bash %} $ tcwallet --recipient [recipient] --amount [amount] {% endhighlight %}

    For example:

    {% highlight bash %} $ tcwallet 71q7t17qcarlg5yxcvfibpzxra 0.003 {% endhighlight %}
